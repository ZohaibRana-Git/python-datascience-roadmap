---

## When to Use What?

| Chart Type  | Use Case                              |
|-------------|---------------------------------------|
| Bar chart   | Categories compare karna              |
| Line chart  | Time series, trends                   |
| Scatter     | Do variables ka relationship          |
| Histogram   | Distribution dekhna                   |
| Boxplot     | Outliers aur spread dekhna            |
| Heatmap     | Correlation matrix visualize karna    |
| Pie chart   | Proportions (max 5-6 categories)      |
