## Key Concepts

- Python me **type declare** nahi karna padta (dynamic typing)
- **Indentation** bahut important hai — spaces se code blocks bante hain
- `print()` se output dikhao, `input()` se user se data lo
- `with` statement use karo files ke liye — auto close hoti hai
- OOP se code reusable aur organized hota hai
