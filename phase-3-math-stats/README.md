## Key Concepts

- **Mean** = average, **Median** = middle value, **Mode** = most frequent
- **Standard Deviation** = data kitna spread hai mean se
- **Normal Distribution** = bell curve — ML me sabse common
- **Correlation** = do variables ka relationship (-1 to +1)
- **Linear Algebra** = ML algorithms ki backbone (matrices, vectors)
