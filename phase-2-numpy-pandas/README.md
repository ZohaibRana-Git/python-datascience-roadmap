## Key Concepts

- NumPy arrays Python lists se **100x faster** hote hain (C me likhe hain)
- Pandas DataFrame = Excel spreadsheet in Python
- `df.head()`, `df.info()`, `df.describe()` — pehle ye 3 hamesha chalao
- Missing values handle karna Data Science ka 60% kaam hai
