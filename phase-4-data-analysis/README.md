## EDA Checklist (Har project me ye karo)

```
1. df.shape          → kitne rows/cols
2. df.info()         → dtypes, nulls
3. df.describe()     → stats summary
4. df.isnull().sum() → missing values
5. df.duplicated()   → duplicates
6. value_counts()    → categorical distribution
7. corr()            → correlations
8. Visualize!        → histograms, boxplots, heatmap
```
