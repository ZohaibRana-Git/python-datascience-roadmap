## Key Concepts

| Term          | Meaning                                          |
|---------------|--------------------------------------------------|
| Neuron        | Ek unit jo input leta hai, output deta hai       |
| Layer         | Neurons ka group                                 |
| Weights       | Model ke learnable parameters                    |
| Activation    | Non-linearity add karta hai (ReLU, Sigmoid)      |
| Backprop      | Error se weights update karna                    |
| Epoch         | Poora dataset ek baar train karna                |
| Batch size    | Ek baar me kitne samples process hote hain       |
| Learning rate | Kitni tezi se weights update hote hain           |
| Dropout       | Overfitting rokne ke liye neurons randomly off   |

---

## Neural Network Architecture

```
Input Layer  →  Hidden Layers  →  Output Layer
   (X)           (learn patterns)    (prediction)
```

---

## When to Use Deep Learning?

- Large dataset (10,000+ samples)
- Complex patterns (images, audio, text)
- Traditional ML kaam na kare
